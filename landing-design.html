Multi-tenant document RAG

Ask your documents. Get answers with citations.

DocIntel is a multi-tenant RAG platform: ingest PDFs, DOCX, and Markdown into a hybrid
lexical + semantic index, then stream cited answers from an LLM. Built on PostgreSQL,
Prisma, and Next.js.

Open dashboard Try the chat

What it actually does

Ingest

Upload a document; it's parsed, chunked, and embedded — ingested records
appear in the dashboard and become queryable.

Retrieve

Hybrid lexical + semantic search with reciprocal-rank fusion. Citations
accompany every answer with the exact passage that was read.

Govern

Per-tenant isolation, Stripe-metered quotas, GUID-stamped audit
log, and a one-click GDPR hard purge of workspace data.

How it works

  1. 01

    Drop a document

    PDF, DOCX, Markdown, TXT, CSV. The system parses and chunks the file at
    query-time semantic boundaries.

  2. 02

    Ask

    A typed query enters the chat. Hybrid retrieval surfaces the most relevant
    chunks across your workspace.

  3. 03

    Get a cited answer

    The model is constrained by an anti-hallucination prompt. Every claim is
    anchored to a chunk ID.
